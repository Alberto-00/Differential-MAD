  Introduction
</h2>
<p dir="auto">
  ðŸ§  The Web DApp <b>KryptoAuth</b> is made with the aim of offering greater security during the authentication operation by exploiting the Ethereum Blockchain technology. The system allows any user to register and wait for a administrator gives him the permissions of "User" or "Admin" to carry out the operation of Login.
